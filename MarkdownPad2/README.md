## MarkdownPad 2 - BitBucket Stylesheet ##

### How to install ###
Open [**MarkdownPad 2**](http://markdownpad.com/) and navigate in the menu to `Tools > Options`.
If the settings popup appears, click on the left tab `Stylesheets`.

#### Free Users ####
Select an already existing stylesheet (for Example `none.css`) and click on the `Edit` button. Insert the Content from `bitbucket.css` and save it.

#### Pro Users ####
Create an new stylesheet and call it `bitbucket.css`. Click on the `Edit` button. Insert the Content from `bitbucket.css` and save it.

### Troubleshootings ###
For best solution, you must change the settings:
- `Tools > Options > Markdown`
  - Markdown processor: **GitHub Flavoured Markdown (Offline)**
  - Document Rendering Mode: **Full Document**
